Para utilizar la plataforma Onecar se deben seguir los siguientes paso:

1. Registrarse en la plataforma correctamente en el siguiente enlace: Registro

Una vez se haya registrado correctamente, se deberá loguear en la web añadiendo el nombre de usuario con el que se registro y la contraseña.

2. Cuando se haya logueado correctamente, será redirigido al enlace para buscar los usuarios que se trasladen al destino que se introduzca. Para realizar esta operación, se debe ingrasar un origen, un destino y una fecha.

Cuando se introduzcan todos los datos, se buscarán los usuarios que se dirijan al destino que haya introducido.

Para reportar algún tipo de problema con la web, debe dirigirse al siguiente enlace Acerca

Será redirigido al enlace con la información de los creadores de la web y además, le podrá enviar un correo electrónico o comunicarse con ellos mediante redes sociales.

Para navegar por los diferentes enlaces de la web, se puede utilizar los enlaces de navegación que se encuentran en la parte superior de cada página.
